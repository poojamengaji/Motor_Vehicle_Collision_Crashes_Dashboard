# Motor_Vehicle_Collision_Crashes_Dashboard
PCDA - Final Project by Pooja Mengaji &amp; Bipika Sapkota
Git repository URL: https://github.com/poojamengaji/Motor_Vehicle_Collision_Crashes_Dashboard

Instruction to run the project
1. Run the Jupyter notebook motor_crashes_eda.ipynb
This file will perform data cleaning and save in the Cleaned_Motor_Vehicle_Collisions_Crashes.csv file in the data/ directory.
Based on the clean data, EDA is performed and various plots will be saved in the plots/ directory.
2. Run app.py as a Streamlit app using VSCode or any IDE.
The app will display a dashboard with menu options. Various visualizations are displayed on each menu.
