## Project Overview
This project analyzes motor vehicle collision data and presents insights through an interactive **Streamlit dashboard**. It includes data cleaning, exploratory data analysis (EDA), and visualization of key patterns such as injuries, contributing factors, and borough-wise trends.

---

## How to Run the Project

### Step 1: Data Cleaning & EDA
Run the Jupyter Notebook: **jupyter notebook motor_crashes_eda.ipynb**

#### Cleans the raw dataset
#### Saves the processed file to: **data/Cleaned_Motor_Vehicle_Collisions_Crashes.csv**
#### Performs exploratory data analysis
#### Generates visualizations stored in: **plots/**

### Step 2: Run the Dashboard

Launch the Streamlit app: **streamlit run app.py**

#### Opens an interactive dashboard in your browser
#### Provides multiple menu options to explore
